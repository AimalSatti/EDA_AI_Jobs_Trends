📊 Overview
This repository contains a cleaned and curated dataset of 1,492 artificial intelligence job postings spanning the years 2025 and 2026. This repository contains a cleaned and curated dataset of 1,492 artificial intelligence job postings spanning the years 2025 and 2026. It provides a detailed look into the evolving AI job market, covering salary benchmarks, required skills, remote work trends, and demand growth across 14 different countries.
📈 Key Statistics
 * Total Records: 1,492
 * Unique Job Titles: 25 (e.g., LLM Engineer, AI Agent Developer, Prompt Engineer)
 * Average Annual Salary: ~$194,557 USD
 * Global Reach: 14 Countries (including USA, UK, Canada, Germany, Singapore, Japan, and more)
   🚀 Potential Use Cases
 * Salary Prediction: Use features like years_of_experience, education_required, and is_llm_role to predict annual_salary_usd.
 * Trend Analysis: Compare demand_score across different job_categories to see which AI fields are growing fastest.
 * Skill Mapping: Parse the required_skills column to identify the most sought-after technologies in 2026.
